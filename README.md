## Tasks

- `student-mark-tracker` - simple Java console app for calculating student mark averages.
- `practical-task-1` - animal shelter console app.
- `practical-task-2` - payment processing console app.
- `m3-practical-task` - smart home controller with custom exceptions and logging.
- `m4-practical-task` - CSV import, collections, analytics, and report export.
- `m5-practical-task` - unit testing practice with JUnit, AssertJ, and Mockito.
- `m7-practical-task` - basic SQL and SQLite database practice.
- `m8-practical-task` - Spring Boot library book tracker API with JPA, H2, DTOs, validation, exception handling, and Basic Auth security.

Each task is kept in its own folder with its own source code and README file.
